Medical Clinics

The medical clinics system is a web application that allows a system of clinics to manage appointments and medical examinations. The application serves multiple user groups, including patients, doctors, nurses, and administrators. Patients can use the application to schedule appointments, make payments, and view their medical records. Doctors and nurses can view and edit the medical records of all patients. During the examination, the medical staff creates a medical report, which is then added to the patient's medical record. Both doctors and nurses have personal working calendars with their working schedules. The administrator manages clinics, examination rooms, doctors and nurses, patient registration requests, medication documents, and more. The system supports payments through credit cards and PayPal using Stripe and a sandbox environment.

Affiliate Bet Admin Panel

The Affiliate Bet admin panel is a web application for a gaming admin panel that supports affiliates, cashiers, and referrals. The application allows users to share banners or links on social networks to promote different resolutions and earn money based on their score. Depending on their role and fees, users can observe different types of reports, including earnings reports, deposits, depositors, wallet balance, and transactions. The application serves multiple user groups, including administrators, managers, affiliates, referrals, and cashiers.

Responsibilities for this project include manual project testing covers design and functionality, Postman API, Microsoft Excel for counting different type of calculations and reporting bugs. Testing is performed in different types of environments.

Property Website

The property website is a web application for publishing, advertising, and booking properties online. The application serves two main user groups: clients and agents and managers. Users can use filters to search for properties based on city, square meters, price, and other criteria. They can send requests for specific properties to agents or managers.

Responsibilities for this project include manual testing, Postman API, and reporting bugs. The version control system used is GitLab. Testing is performed in different types of environments.

Healthcare Services

The healthcare services platform is a web platform where users can find information about healthcare services offered by medical institutions across the country. The application serves multiple user groups, including users, entity representatives, entities, moderators, and administrators. Users can search for experiences from various clinics, hospitals, and doctor's offices, filtered by keywords such as doctor names, diseases, or specialties. They can make appointments or calls for a specific entity with a paid plan. The platform allows users to recommend, vote, and report on their experiences. Entity representatives can manage information for entities, manage roles for representatives, and respond to experiences.

Responsibilities for this project include frontend development, manual testing, Postman API, reporting bugs, and automation testing in Cypress. Testing includes writing end-to-end, functional, regression tests, and using Git, GitHub, and CI/CD. The project includes testing the web application and admin panel in different types of environments.
